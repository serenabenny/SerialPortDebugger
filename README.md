<b>功能概要：</b><br/>
1.串口，TCP服务端/客户端，UDP服务端/客户端通讯调试<br/>
2.支持Hex,ASCII,UTF8,GB2312编码调试<br/>
3.接收数据可通过右键菜单一键计算Hex对应的数值<br/>
4.字节计算器/检验值计算器（CRC16,自定义多项式CRC16，XOR)<br/>
5.串口超级终端<br/>
6.Socket通信监视器<br/>
7:Ping调试<br/>

<b>计划更新：</b><br/>
1.支持SQLITE数据库，能保存调试命令及最后使用的通信参数信息等<br/>
2.调试命令的导入和导出<br/>
